# Nunes Sports

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/stefanietds/NunesSports-App.git
```

Entre no diretório do projeto

```bash
  cd NunesSports-App
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```


## Stack utilizada

**Front-end:** React
